# ScholarLink — Faculty Finder

AI-powered research matchmaking for Shahjalal University of Science and
Technology (SUST). Describe a research interest in plain language and get
matched with the professors and PhD researchers most relevant to it —
semantic search over 611 scraped faculty profiles.

This Space mirrors the `deploy/railway` combined-container build, adapted
for Hugging Face's fixed uid-1000 container runtime.
